# JobSpark – Django Job Portal

A fully functional job portal built with **Django 4+**, **Bootstrap 5**, and **SQLite**.
Converted from a vanilla HTML/CSS/JS frontend — same UI, full backend power.

---

## Quick Start

## 🚀 Features

* 🔍 Search and filter jobs by title, location, category, and job type
* 📝 Apply for jobs with a simple application form
* 💼 Recruiter dashboard to post, edit, and delete job listings
* 📊 User dashboard to track applied and saved jobs
* ⭐ Save/bookmark jobs for later
* 🔐 User authentication (Login, Register, Logout)
* ⚡ Fast and responsive UI

---

## 🧩 General

* 🌙 Dark mode toggle (persisted using localStorage)
* 🔔 Bootstrap toast notifications for all user actions
* 🛠️ Django admin panel for full data management
* 📱 Fully responsive design (mobile-friendly)
* 🔗 Django messages integrated with toast notifications (messages → toast bridge)

---

## 🛠️ Technologies Used

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** SQLite (default)
* **Deployment:** Render
* **Server:** Gunicorn
* **Static Files Handling:** WhiteNoise
